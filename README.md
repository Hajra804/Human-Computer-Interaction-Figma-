# Human-Computer-Interaction-Figma-
# Application Prototype

## Overview

This project was developed as part of the **AHCI Semester Project (Spring 2026)**. The objective was to design a complete, user-centered application prototype using **Figma**, following the core principles of Human-Computer Interaction.

The prototype focuses on providing an intuitive, visually consistent, and user-friendly experience while demonstrating modern UI/UX design practices.

---

# Project Objectives

* Design a complete application prototype using Figma.
* Apply Human-Computer Interaction (HCI) principles.
* Create a user-friendly and accessible interface.
* Develop a consistent and visually appealing UI/UX.
* Build an interactive prototype with smooth navigation between screens.

---

# Features

The prototype includes:

* Splash Screen
* Login Screen
* Sign Up Screen
* Home Screen
* Core Functional Screens
* Navigation Flow
* Interactive Prototyping

---

# HCI Principles Applied

The design incorporates the following Human-Computer Interaction principles:

* User-Centered Design
* Usability and Learnability
* Consistency and Standards
* Feedback and System Response
* Error Prevention and Handling
* Accessibility Considerations
* Visual Hierarchy
* Modern Layout and Navigation

---

# Design Highlights

* Clean and modern user interface
* Consistent color palette and typography
* Reusable components and design elements
* Proper spacing, alignment, and layout structure
* Interactive navigation between screens
